#  钓鱼线

##  GetFishingLineColor

服务端客户端

###  服务端接口

method in mod.server.component.fishingLineCompServer.FishingLineCompServer

-

描述

获取服务端当前设置的钓鱼线颜色

-

参数

  | 参数名 | 数据类型 | 说明 |
| --- | --- | --- |
   | itemName | str | 鱼竿物品的完整名称，如 "minecraft:fishing_rod" 或自定义鱼竿名称 |

-

返回值

 | 数据类型 | 说明 |
| --- | --- |
  | tuple(float,float,float) | rgb颜色值，每个分量0.0-1.0 |

-

示例

```python
import mod.server.extraServerApi as serverApi
comp = serverApi.GetEngineCompFactory().CreateFishingLine(playerId)
color = comp.GetFishingLineColor("mymod:custom_fishing_rod")
print color

```

##  SetFishingLineColor

服务端

method in mod.server.component.fishingLineCompServer.FishingLineCompServer

-

描述

设置钓鱼线的颜色，设置后会自动同步到所有客户端。

-

参数

  | 参数名 | 数据类型 | 说明 |
| --- | --- | --- |
   | itemName | str | 鱼竿物品的完整名称，如 "minecraft:fishing_rod" 或自定义鱼竿名称 |
   | color | tuple(float,float,float) | 鱼线颜色rgb，取值[0, 1] |

-

返回值

 | 数据类型 | 说明 |
| --- | --- |
  | bool | 是否设置成功 |

-

示例

```python
import mod.server.extraServerApi as serverApi
comp = serverApi.GetEngineCompFactory().CreateFishingLine(playerId)
comp.SetFishingLineColor("mymod:custom_fishing_rod", (1.0, 0.0, 0.0))

```

##  SetFishingLineMax

服务端

method in mod.server.component.fishingLineCompServer.FishingLineCompServer

-

描述

设置钓鱼线的最大长度（超过此长度会断线），设置后会自动同步到所有客户端。

-

参数

  | 参数名 | 数据类型 | 说明 |
| --- | --- | --- |
   | itemName | str | 鱼竿物品的完整名称，如 "minecraft:fishing_rod" 或自定义鱼竿名称 |
   | maxLength | float | 最大长度，单位为格，钓鱼竿默认值为32.0 |

-

返回值

 | 数据类型 | 说明 |
| --- | --- |
  | bool | 是否设置成功 |

-

示例

```python
import mod.server.extraServerApi as serverApi
comp = serverApi.GetEngineCompFactory().CreateFishingLine(playerId)
comp.SetFishingLineMax("mymod:custom_fishing_rod", 64.0)

```
